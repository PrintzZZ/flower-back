# 基于vue的鲜花电商app
## 更新中...

> 项目预览地址 



这是一个基于vue的电商平台的后端部分

## 运行方式
>克隆项目

`git clone git@github.com:PrintzZZ/flower-back.git`

>安装依赖

`npm install`

>打开mongodb数据库连接池

`./mongo  --dbpath d:\data\db`

>连接数据库

`./mongo`
>创建名为flowershop的数据库
`use flowershop`

>运行

`npm run dev`

## 使用技术栈

>使用express创建项目脚手架

>使用node.js编写接口

>使用mongodb作为数据库
