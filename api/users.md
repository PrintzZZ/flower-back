# 1.用户注册接口

>接口地址 : http://localhost:3000/users/register

>请求方式 : post

>参数 : 用户名username 密码password 手机号tel

>返回结果 : { code: '' ,message: '', data: ''}

>返回参数说明:

|参数名称|说明|
|----|----|
|10001|注册成功|
|10002|用户已存在,请直接登录|

# 2.用户登录接口

>接口地址 : http://localhost:3000/users/login

>请求方式 : post

>参数 :  密码password 手机号tel

>返回结果 : { code: '' ,message: '' ,data: '',token: ''}

>返回参数说明:

|参数名称|说明|
|----|----|
|10003|用户不存在|
|10004|登录成功|
|10005|密码错误|

# 3.修改用户信息接口

>接口地址 : http://localhost:3000/users/update

>请求方式 : post

>参数 :  用户名newname, 手机号tel, 性别sex(字符型), 生日birth

>返回结果 : { code: '' ,message: ''}

>返回参数说明:

|参数名称|说明|
|----|----|
|10006|用户不存在,修改失败|
|10007|修改用户信息成功|

>tip: 使用tel查询用户,查不到返回10006; 可供用户修改用户名,性别,生日; 手机号为注册账户无法修改
